Factory Robot MVP

基于 ROS2 Humble + Gazebo Ignition 的工厂移动机器人仿真全栈项目 —— 从 SLAM 建图、自主导航、YOLO 视觉检测到机械爪控制，Docker 容器化隔离部署

GitHub 仓库 查看演示 架构设计

演示视频

视频内容: 一键启动仿真 → SLAM 建图 → 键盘遥控 → 保存地图 → Nav2 自主导航 → YOLO 检测 → 夹爪控制

核心功能

🗺️

SLAM 在线建图

slam_toolbox 异步在线模式，键盘遥控机器人探索厂房，实时构建 2D 栅格地图，支持地图保存与重载

slam_toolbox
🧭

Nav2 自主导航

AMCL 定位 + NavFn 全局规划 + DWB 局部规划 + 行为树恢复策略，在 RViz 中点击目标点即可自主导航避障

Nav2
👁️

YOLO 目标检测

Docker 容器内运行 YOLO 推理节点，订阅相机图像发布 Detection2DArray 标准消息，Timer 解耦采集与推理避免背压

Ultralytics
🤖

1-DOF 平行夹爪

双 prismatic 关节 + mimic 联动，ros2_control 位置控制器驱动，Topic 指令控制开合（0.0~0.04m）

ros2_control
🏭

厂房仿真场景

Gazebo Ignition Fortress 加载 SDF 厂房世界，包含墙壁、货架、货物箱，差速驱动机器人带 LiDAR + RGB-D 相机

Gazebo Ignition
🐳

Docker 容器化部署

Split Architecture: Gazebo 宿主机运行规避 GPU 转发问题，计算节点容器化通过 DDS 主机网络通信

Docker Compose

数据流

Topic类型方向说明
/scanLaserScanGazebo → ROS22D 激光扫描，SLAM/Nav2 输入
/camera/color/image_rawImageGazebo → ROS2彩色图像，YOLO 输入
/camera/depth/image_rawImageGazebo → ROS2深度图像，避障/点云
/cmd_velTwistROS2 → Gazebo速度指令 (Nav2/遥控)
/odomOdometryGazebo → ROS2里程计
/mapOccupancyGridSLAM → ROS22D 栅格地图
/vision/detectionsDetection2DArrayYOLO → ROS2目标检测结果
/gripper_controller/commandsFloat64MultiArrayROS2 → Controller夹爪位置指令
